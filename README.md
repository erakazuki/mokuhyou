# 筋トレ進捗共有アプリ

彼女と筋トレの進捗を共有するシンプルな Django アプリです。

## 機能

- **ログイン / 新規登録** — ユーザー認証が必須
- **共有カレンダー** — 秘密の URL を知っている人だけが同じカレンダーを閲覧・編集
- **筋トレ記録** — 日付をクリックして ON/OFF（色付きで可視化）
- **目標設定** — 週あたりの筋トレ回数を設定

## セットアップ

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

ブラウザで http://127.0.0.1:8000/ を開いてください。

## 使い方

1. 新規登録してログイン
2. 「新しいカレンダーを作成」で共有カレンダーを作成
3. 表示された URL を相手に共有（相手もアカウント作成・ログインが必要）
4. カレンダー上の日付をクリックして筋トレ記録
5. 「目標設定」で週の目標回数を設定

## URL 構成

| パス | 説明 |
|------|------|
| `/` | ホーム（カレンダー作成・参加） |
| `/login/` | ログイン |
| `/signup/` | 新規登録 |
| `/c/<slug>/` | 共有カレンダー |
| `/c/<slug>/goals/` | 目標設定 |
